# X-like Project

## English
This project was built with **Next.js** just for fun and for educational purposes.  
It uses the external layout inspired by the **X (ex-Twitter)** website.

The styling is powered by **Tailwind CSS**, which allowed for rapid development and easy customization of the UI components.

You can explore the project, learn from the code, and experiment with Next.js features like SSG, ISR, and API fetching.

### How to run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/X-learn-project.git
   ```

2. Go to the project folder:
   ```bash
   cd X-learn-project
   ```

3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```
   or
   ```bash
   yarn dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to see the project.
